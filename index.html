Use the arrow keys to pan the map. Hold Ctrl to instead move the yellow cursor around.

To execute code, put hyphens on the top edge of your code block, pipes on the left edge, and finally an asterisk at the top-left corner. the World of Code uses a stack-based language. You can access memory by reading from cells.

  • Number literals from 0 - 255 by representing as hexadecimal (note – must be uppercase!)
  • a b +; a b -; a b *; a b / (integer division); a b % (spaces are completely optional)
  • a b > and a b = to compare
  • : duplicates the topmost element; . deletes it
  • X exchanges the topmost 2 elements; a Y moves to the top the element buried by a others.
  • a W does the opposite.
  • c x y ? jumps to (x, y) if c is true.
  • x y G gets the character at (x, y). x y n P places a character (returning whether this succeeded). Note that only printable ASCII characters may be placed.
  • x y ^ gets the offset from the current location of execution. x y $ gets the offset from the starting location.
  • ! gives the logical negation of the topmost element (0 turns into 1; any other value turns into 0)
  • Any lowercase letter can be used as a variable. To assign the topmost stack element to one, use # followed by the name.
